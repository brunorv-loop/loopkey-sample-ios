  • LKCommState

    Describes the current state of the LKCommunicationManager instance.

    • unknown: state is still unknown, probably because manager hasn’t been initialized yet.
    • enabled: manager is enabled and communicating with devices.
    • disabled: manager is disabled, probably because BLE is turned off.
    • unsupported: BLE isn’t supported on device.
